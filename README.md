# medicore-hospital-management-system
 Medicore is a Python-based Hospital Management System with role-based portals for patients, doctors &amp; admins. Patients book appointments &amp; access AI disease prediction (under dev). Doctors manage patients &amp; diagnoses. Admins add doctors &amp; generate reports. Uses Pandas, scikit-learn &amp; CSV storage. Secure, modular &amp; educational.
